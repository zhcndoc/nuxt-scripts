[Meta Pixel](https://www.facebook.com/business/tools/meta-pixel) 让你能够衡量、优化并为你的 Facebook 广告活动建立受众。

Nuxt Scripts 提供了注册脚本组合函数 [`useScriptMetaPixel()`{lang="ts"}](/scripts/meta-pixel){lang="ts"}，方便你在 Nuxt 应用中集成 Meta Pixel。
