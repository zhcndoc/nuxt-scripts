[Snapchat 像素](https://businesshelp.snapchat.com/s/article/snap-pixel-about){:target="_blank"} 可帮助你衡量 Snapchat 广告活动在跨设备上的影响。

Nuxt Scripts 提供了一个注册脚本组合函数 [`useScriptSnapchatPixel()`{lang="ts"}](/scripts/snapchat-pixel){lang="ts"}，可以轻松地在你的 Nuxt 应用中集成 Snapchat 像素。
