[Segment](https://segment.com/) 让你收集、清理并控制你的客户数据。Segment 帮助你了解客户并个性化他们的体验。

Nuxt Scripts 提供了一个注册脚本组合函数 [`useScriptSegment()`{lang="ts"}](/scripts/segment){lang="ts"}，以便你轻松地在 Nuxt 应用中集成 Segment。
