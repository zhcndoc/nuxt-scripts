[X Pixel](https://x.com/) 让你收集、清理并控制你的客户数据。X 帮助你了解客户并个性化他们的体验。

Nuxt Scripts 提供了一个注册脚本组合函数 [`useScriptXPixel()`{lang="ts"}](/scripts/x-pixel){lang="ts"}，可以轻松地在你的 Nuxt 应用中集成 X Pixel。
