## 快速开始

要开始，只需运行：

```bash
npx nuxi@latest module add scripts@beta
```

> [!TIP]
> 使用 [skilld](https://github.com/harlan-zw/skilld) 为此包生成一个代理技能：
> ```bash
> npx skilld add @nuxt/scripts
> ```

就是这样！Nuxt Scripts 模块将被下载并添加到您的 Nuxt 配置的 `modules` 中。
