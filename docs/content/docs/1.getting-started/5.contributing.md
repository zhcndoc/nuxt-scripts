## 准备你的开发环境

`@nuxt/scripts` 是一个使用 pnpm 工作区的单体仓库。用于安装和链接依赖的包管理器必须是 pnpm。

克隆仓库后，你可以通过以下命令准备你的环境：

```sh
pnpm i && pnpm dev:prepare
```

## 运行示例环境

你可以通过运行开发服务器来体验 playground：

```sh
pnpm dev
```
